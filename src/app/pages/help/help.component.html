System Help Guide

1. Login to the System

Staff members must log in using their username and password. After successful authentication, users will be redirected to the dashboard.

2. Guest Management
  • Search guests using phone number or email address.
  • Add new guests using the Add Guest button.
  • Update guest information when necessary.
3. Room Management
  • View all available rooms in the system.
  • Add new rooms using the Add Room button.
  • Edit room details such as bed configuration and price.
  • Delete rooms if they are no longer available.
4. Creating Reservations
  • Search for the guest using phone number or email.
  • Enter the check-in and check-out dates.
  • Check available rooms for the selected dates.
  • Select a room and confirm the reservation.
  • After confirmation, the system automatically generates a reservation invoice.
5. Reservation Invoice

After creating a reservation, the system generates a PDF invoice. Staff can print or download the invoice for the guest.

6. Canceling Reservations

Reservations can be canceled using the reservation number. When canceled, the reservation will be marked as inactive in the system.

Staff can retrieve reservation details using the reservation ID and verify the guest before canceling.

7. Dashboard Search Tools

The dashboard provides quick search tools to assist hotel staff.

  • Room Number Search: Allows staff to check whether a room is currently occupied or available.
  • Reservation Number Search: Allows staff to retrieve reservation details quickly for verification, modification, or cancellation.
8. Future Enhancements
  • Dashboard search for room occupancy status.
  • Quick reservation modification tools.
  • Email notifications for reservation confirmations.
  • Automated reports for daily revenue and occupancy.
